# Setup Instructions

## Prerequisites

Before running this project, ensure you have Node.js and npm installed on your system.

## Installation Steps

1. **Install Node.js**: Download and install Node.js from [nodejs.org](https://nodejs.org/)

2. **Install Dependencies**: Run the following command in the project root:
   ```bash
   npm install
   ```

3. **Verify Installation**: Check that all dependencies are installed correctly:
   ```bash
   npm run test
   ```

## Dependencies Included

### Production Dependencies
- `react` ^18.2.0 - React library
- `react-dom` ^18.2.0 - React DOM rendering
- `axios` ^1.6.0 - HTTP client for API calls
- `recharts` ^2.8.0 - Chart library for data visualization
- `lucide-react` ^0.294.0 - Icon library

### Development Dependencies
- `@vitejs/plugin-react` ^4.1.0 - Vite React plugin
- `typescript` ^5.2.2 - TypeScript compiler
- `tailwindcss` ^3.3.0 - CSS framework
- `jest` ^29.7.0 - Testing framework
- `@testing-library/react` ^13.4.0 - React testing utilities
