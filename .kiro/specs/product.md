# Product Specification: Cosmic Crypto Dashboard

## Goal
Build a React dashboard that visualizes the "correlation" between Bitcoin's price and the number of Asteroids (Near Earth Objects) passing Earth.

## Core Features
1. **Header:** "Cosmic Crypto Tracker" with a refresh button.
2. **Key Metrics (Cards):**
   - **Bitcoin Price:** Live price in USD.
   - **Asteroid Count:** Total Near Earth Objects (NEOs) today.
   - **Hazard Score:** Percentage of asteroids marked "potentially hazardous."
3. **Main Chart:** A Line Chart displaying the last 7 days.
   - Line 1 (Green): Bitcoin Price.
   - Line 2 (Red): Number of Asteroids.
4. **Theme:** Dark mode, space-themed (black/purple backgrounds).
